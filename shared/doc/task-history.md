# タスク実施履歴

## プロジェクト概要

プロジェクトセカイ（プロセカ）の楽曲マスタデータとアーティストマスタデータを管理するWebアプリケーション。Vue 3、TypeScript、Tailwind CSSを使用し、既存のREST APIを利用してCRUD操作を提供する。
